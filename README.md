## Usage

### Git Clone
```bash 
git clone https://github.com/richeyphu/MyWarehouse-Project.git
```

### Required Packages
- `PyQt5`
- `PyQt5-stubs`
- `pyqt5-tools`


## Development Tools
* **IDE :** `PyCharm Professional 2021.2 or newer`
* **Interpreter :** `Python 3.9`
* **GUI Framework :** `PyQt5 with Qt Designer`
* **DBMS :** `SQLite3`
